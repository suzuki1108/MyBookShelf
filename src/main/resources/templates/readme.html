このアプリケーションについて

このアプリケーションは、制作者のエンジニア学習の成果物として作成されたものです。

個人の制作物ですので、個人を特定できる情報などの保存はご遠慮ください。

※いたずらや、悪い使いかたもしないでください。


アプリケーションの概要

このアプリケーションは、制作者が本を買うときに、「あれ？この本持ってなかったっけ？」

と疑心暗鬼になったことから生まれた本棚管理アプリケーションです。

検索窓から本を検索し、本棚に追加することで、自分だけの本棚を作成することができます。

※本が読めるわけではありません。

現在の機能は本の検索、追加、削除、本棚の追加のみですが、

今後は他のユーザの本棚を覗きに行けるような機能も追加しようと思っています。
